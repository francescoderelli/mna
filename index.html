Confronto premi RA

Tutto gira nel browser. I file non vengono caricati online.

Dove prendere i file

Premi_Mensili_x_Sede (xlsx)
Dashboard tecnica → Premi → tab Grafici → seleziona la sede → vai in fondo in “Premi Mensili x Sede e Tipo Risorsa” → estrapola Excel.

Sum_of_Importo (xlsx)
Dashboard direzionale → Obiettivo sedi → tab P&L Gestionale → selezionare sede → impostare data di inizio 01/01/2025 → estrapolare Excel con i risultati dei vari mesi.

Sum_of_Obiettivoprev (xlsx)
Dashboard direzionale → Obiettivo sedi → tab Tabella riepilogo → selezionare sede (attenzione: nomi sede diversi rispetto al P e L) → andare nel secondo riquadro dove c’è scritto “Gli obiettivi vengono conteggiati al più fino al mese in corso compreso” → estrapolare Excel.

Tip: i file devono essere in formato .xlsx e scaricati direttamente dal gestionale (non “copiati/incollati” in altri Excel).

⚠️ Nota importante sui nomi sede I nomi delle sedi tra P&L Gestionale e Tabella riepilogo possono essere diversi. Se selezioni due sedi “equivalenti” ma con nomi non allineati, il sistema può bloccarsi con errore di coerenza. In quel caso, riesporta i file assicurandoti che la sede scelta sia quella corretta in entrambi i report.
